# Flask Todo App

シンプルなTodoリストアプリケーション

## 機能
- タスクの追加
- タスクの削除
- タスクの一覧表示

## セットアップ

1. リポジトリをクローン
```bash
git clone https://github.com/Takada-machine12/webapp.git
cd webapp
```

2. 仮想環境を作成
```bash
python -m venv venv
source venv/bin/activate  # Windowsの場合: venv\Scripts\activate
```

3. 依存関係をインストール
```bash
pip install -r requirements.txt
```

4. アプリを実行
```bash
python app.py
```

5. ブラウザで http://127.0.0.1:5000 にアクセス
